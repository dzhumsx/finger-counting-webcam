# Finger Counting Webcam

This is a simple computer vision project in Python that uses the webcam to perform real-time hand and pose tracking, focusing on counting raised fingers. The project relies on the OpenCV and MediaPipe libraries.

## Features
- Hand Tracking with real-time finger counting.
- Pose Estimation.
- Frames Per Second (FPS) display on the screen.

## Technologies and Libraries
- Python (3.12.0)
- OpenCV
- MediaPipe

## How to Run

1. Ensure you have the dependencies installed:
```bash
pip install opencv-python mediapipe
```

2. Run the main script:
```bash
python main.py
```

3. A window will open displaying your webcam feed. To close the application, press the "q" key or close the window.
